Production API Integration Guide

1. Google Maps Journey Sharing API

For real-time asset (technician) movement and accurate traffic-aware ETAs:

  • Ensure your Google Cloud Console has Mobility Services / Fleet Engine enabled.
  • Load the script using the library hook: journeySharing.
  • Instantiate a google.maps.journeySharing.JourneySharingSession specifying the trackingId (booking index).
  • Update latitude/longitude coordinates from your driver-side app to Fleet Engine. The maps consumer view updates reactively!

2. WhatsApp Business Cloud API & Supabase AI

Build a production webhook answering questions powered by OpenAI:

  • Set up a Supabase Edge Function running Deno to act as the public webhook endpoint.
  • Verify the Webhook inside the Meta App Developer Dashboard with your custom Verification Token.
  • When a user sends an inbound message, Supabase routes the string to the OpenAI GPT-4o-mini SDK.
  • Pass the generated text answer back to Meta's Send Message REST API: POST https://graph.facebook.com/v18.0/{phone-number-id}/messages
